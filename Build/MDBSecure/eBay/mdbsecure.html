What is MDBSecure?

  • An MS Access Security utility, providing protection for your databases.
  • MDBSecure follows the recommended approach to securing Microsoft® Access databases, it will encrypt your access database and remove security weaknesses.


    • SecFaq - Security FAQ (Recommended approach by Microsoft)

  • Although nothing is ever 100% secure in computing, following this approach will make an MS Access Database more secure.
  • For more information about the theory behind securing MS Access databases please see the SECFAQ (security FAQ) document.


    • MS Access Security weakenesses

  • Different versions of mdb's have different weaknesses.
  • The mdb is designed to be easy to access, however there are a lot security features available within access which can be fined tuned.
  • MDBSecure uses the built in encryption provided by MS Access.





    • Features
  • Removes Admin user permissions
  • Assigns new super user
  • Encrypt database
  • MS Access 2000/2002 format
  • Workgroup creation


    •
  • Append tables from another DB
  • Creates Jet Connection Strings
  • Opens Secure Databases with MS Access
  • Check for Updates
  • Easy to use, help provided

    Without wanting to give away a blue print for someone else to write their own program, I can provide a set by step process of the manual process.
     
    1.Create new workgroup, Join it
     
    2.Open up a database or create new one
     
    3.Create Super User with password
     
    4.Log out / then login in with it
     
    5.Create new database, Add passwords for Admin
     
    6.Start modifying permissions, Remove them from Admin and Admins Group
     
    7. Now Encrypt
     
    8. Add new groups and permissions
     
    9. Import tables etc from master database, ownership will have changed.
